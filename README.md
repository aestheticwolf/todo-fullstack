# Full-Stack To-Do List App

A clean, responsive, and fully functional To-Do List application built with React, Node.js, and TypeScript. This app lets you manage your tasks efficiently  add, update, delete, and organize them.

# Features

- Add new tasks - with a title and description  
- Mark tasks as Complete or Incomplete - with one click  
- Edit task name and description -  directly in the list  
- Delete tasks - with a confirmation prompt  
- View full task details - in a dedicated page  
- Filter tasks - by status: All, Complete, or Incomplete* 
- Sort tasks - by name or status  
- Search tasks - by name in real time  
- Responsive design - works  on mobile, tablet, and desktop

# Built With

-  Frontend : React + TypeScript + Custom CSS  
-  Backend : Node.js + Express + TypeScript  
-  API : RESTful endpoints (GET, POST, PUT, DELETE)  
-  Deployment :  
  - Frontend: [Netlify](https://www.netlify.com/)  
  - Backend: [Render](https://render.com/)
